Hard Coded Values

This are what you see when setting the height or width of something. Like widht: 200px for example. So the use of "px" is actually bad since its not dynamic and it makes your element unresponsive.

So the best way to do this is to use "Percetage". [width: 100%] and then use the "px" value on "max-width" like:

width: 100% and max-width: 400px

This makes the element responsive but with the use of the max-width you can set it how ever you like.
